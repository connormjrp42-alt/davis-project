Создание Discord-бота

Соберите готового бота для семьи или фракции. Настройки сохраняются в облаке и синхронизируются через сайт без ручной работы с файлами.

Руководство по созданию

Короткий путь, чтобы бот сразу заработал без ошибок.

  1. Создайте приложение в Discord Developer Portal. В разделе Bot нажмите Add Bot, затем скопируйте токен.
  2. Подготовьте доступы. Скопируйте Application ID (это CLIENT_ID) и при желании ID сервера (GUILD_ID) для быстрой регистрации команд.
  3. Создайте проект через плитку + Создать бота. Заполните поля и нажмите кнопку сохранения настроек.
  4. Подключите облачный режим. Укажите BOT_TOKEN и нажмите Подключить бота. Файлы скачивать не нужно.
  5. Проверьте в Discord. Нажмите Синхронизировать и Опубликовать эмбед, чтобы проверить доступ к каналу.

Важно: никому не передавайте BOT_TOKEN. Если токен утек — сразу сбросьте его в Developer Portal.

Управление ботами

Нажмите + Создать бота, чтобы добавить плитку. Конструктор открывается только при переходе в плитку конкретного бота. Список ботов хранится на сайте и привязан к вашему Discord-аккаунту.
